олная цепочка:

Replit (auto-commit) → [вручную] → GitHub → [git pull] → локальная машина

Когда хочешь обновить локалку:

В Replit Shell:
git push github main

На локальной машине:
git pull

Когда хочешь отправить изменения с локалки на Replit:

На локальной машине:
git add .
git commit -m "описание изменений"
git push

В Replit Shell:
git pull github main

Шаг с git push github main в Replit Shell — единственное что нужно делать вручную. Всё остальное (git pull на локалке) — стандартный git.



Запуск поиска параметров бота
Из папки bot/:

cd C:\DATA\bots\replit_scalper\bot

Базовый запуск (параметры из config.yaml):

python optimizer.py

С указанием пары, периода и таймфрейма:

python optimizer.py --symbol BTCUSDT --timeframe 5m --start 2024-01-01 --end 2024-04-01

Больше триалов = точнее результат:

python optimizer.py --symbol BTCUSDT --timeframe 5m --start 2024-01-01 --end 2024-04-01 --trials 300

Все опции:

python optimizer.py --help

Опция	По умолчанию	Описание
--symbol	из config.yaml	Торговая пара
--timeframe	из config.yaml	Таймфрейм (1m, 5m, 15m…)
--start	из config.yaml	Начало периода
--end	из config.yaml	Конец периода
--trials	100	Количество попыток
